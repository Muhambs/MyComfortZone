READ-ME:
1.Install Python 3.12
2.Download and open Pycharm app and open the project on the app.
3.Install Django in the Pycharm terminal using the command: pip install django
4.Install crispy-forms in the Pycharm terminal using the command: pip install django-crispy-forms
5.Install pillow in the Pycharm terminal using the command: pip install pillow
6.Migrate the project using the command 'cd/'
7.Run the project using the command: python manage.py runserver
